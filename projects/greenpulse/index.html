AI-Powered Urban Green Space Analyzer

A full-stack Django application that uses Google's Gemini AI to provide deep, comparative insights into the green spaces of cities worldwide.

View on GitHub See Screenshot

Key Features

Generative AI Core

Uses Google's Gemini AI to generate rich, narrative analysis, overcoming the limitations of traditional web scraping.

Dual City Comparison

Provides a direct, side-by-side comparison of two cities, highlighting key metrics for easy analysis.

Interactive Geospatial Data

Fetches live data from OpenStreetMap to calculate total city area and render interactive maps with Folium.

Technology Stack

Python Django Generative AI OSMnx Folium Bootstrap 5
