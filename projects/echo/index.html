Echo.
Live Project Source Code

Listen to the World.

Echo is an interactive web app to explore and tune into live radio stations from across the globe.

Core Features

  • Interactive World Map: Explore stations on a beautiful, dark-themed CartoDB map.
  • Live Audio Streaming: Click any custom-styled marker to play live radio directly.
  • Real-time Search: Instantly filter hundreds of stations by name or country.
  • Automated Data Seeding: Backend script automatically populates the database.
