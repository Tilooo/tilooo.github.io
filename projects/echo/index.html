-Echo-

An interactive web application to explore and listen to live internet radio stations from around the world.

View Live Project View Source Code

Features

  • Interactive World Map: Explore stations on a beautiful, dark-themed CartoDB map.
  • Live Audio Streaming: Click any custom-styled marker to play live radio directly.
  • Real-time Search: Instantly filter hundreds of stations by name or country.
  • Automated Data Seeding: Backend script automatically populates the database from a public API.

Technology Stack

Python Django Django Rest Framework React.js Vite Leaflet React-Leaflet Tailwind CSS SQLite
