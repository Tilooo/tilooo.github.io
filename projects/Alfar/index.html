A refined digital sanctuary for handcrafted artifacts. Built with "slow ceramics" philosophy at its core, combining artisanal textures with modern interactive solutions.

Core Stack Django & Vanilla CSS
Features L10n, AJAX, Auth
Status Production Ready

Artisanal Vision

The project mimics the organic forms of clay. Every interaction—from the smooth auto-scrolling product sliders to the custom sidebar navigation—is designed to feel intentional and fluid.

Dynamic Masonry Sliders

Curated selections of Mugs, Vases, and Decor components presented through a seamless, infinite horizontal scroll.

Multilingual (i18n)

Fully bilingual architecture (ES/EN) across models and templates, ensuring an international reach for local craft.

The Engineering

Beyond aesthetics, Alfar is a robust web application featuring secure transaction states and persistent user experiences.

Interactive Cart Logic

Optimized UX with +/- quantity management and automated stock updates through protected Django views.

Workshop Booking System

A sophisticated class-management portal with real-time seat availability tracking and pricing integration.

Sidebar AJAX Auth

Custom login/registration systems integrated into a slide-out drawer, allowing for friction-less conversions.
