Find The Flow

A powerful, personal job listings aggregator built with Python and Django that scrapes, stores, and displays job data from across the web.

Project Overview

Find The Flow (Aggre) was born from the need to simplify the job search process. Instead of visiting multiple job boards daily, this application automates the task by scraping data from various sourcesâ€”including complex, JavaScript-driven websites.

The project showcases a full-stack development process, from initial scraping experiments in Jupyter Notebooks to building a robust Django backend, a dynamic front-end with search and pagination, and overcoming real-world challenges like anti-bot measures. It's a practical tool and a comprehensive demonstration of modern web development and data scraping techniques.

Key Features

Multi-Source Scraping

Aggregates data from both static sites (with Requests & BeautifulSoup) and dynamic, JS-heavy sites (with Selenium).

Dynamic Search & Filter

A powerful, responsive search bar allows users to instantly filter jobs by keywords in the title or company name.

Database Persistence

Uses a SQLite database via the Django ORM to store data, preventing duplicates and ensuring data integrity.

Polished User Interface

A stylish, responsive UI with pagination, clean typography, and a modern "glassmorphism" design aesthetic.

The Journey & Challenges

This project was a deep dive into practical problem-solving. The biggest challenge was overcoming the anti-scraping measures on modern websites. Initially, `requests` failed on sites like RemoteOk and Stack Overflow because their content is loaded dynamically with JavaScript.

The solution was to pivot to **Selenium**, which automates a real browser. However, this introduced new challenges, including "human verification" (CAPTCHA) checks. By running Selenium in a visible mode, we diagnosed the issue and made the strategic decision to pivot to a more scraper-friendly target (RemoteOk) to keep the project on track. This iterative process of diagnosing, strategizing, and implementing a new solution was the most valuable learning experience of the project.
