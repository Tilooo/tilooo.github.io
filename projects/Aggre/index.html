Project: Aggre

A personal job listings aggregator that scrapes, stores, and displays job data from across the web.

Watch Demo View on GitHub

Project Overview

Search Aggregator born from the need to simplify the job search. Instead of visiting multiple job boards daily, this application automates the task by scraping data from various sourcesâ€”including complex, JavaScript-driven websites.

The project showcases a full-stack development process, from scraping experiments to a robust Django backend and a dynamic front-end. It's a practical tool and a demonstration of modern web development and data scraping techniques.

Key Features

Multi-Source Scraping

Aggregates data from both static sites (Requests & BeautifulSoup) and dynamic sites (Selenium).

Dynamic Search & Filter

A powerful search bar allows users to instantly filter jobs by keywords in the title or company.

Database Persistence

Uses a SQLite database via the Django ORM to store data, preventing duplicates and ensuring integrity.

Polished User Interface

A stylish, responsive UI with pagination, clean typography, and a modern "glassmorphism" design.
