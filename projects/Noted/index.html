About This Project

“Noted” was a comprehensive journey into full-stack web development. The goal was to build a complete, production-ready e-commerce application from scratch, focusing on a beautiful user interface and a robust back-end. The project covers the entire customer journey — from browsing products in a dynamic carousel, to a seamless slide-out cart experience, and finally a secure checkout powered by Stripe.

Key Features

  • Glassmorphism UI: A stunning “frosted glass” design with a custom yellow-and-green theme.
  • Full E-commerce Flow: Product listings, detail pages, search, categories, and checkout.
  • Interactive Cart: A slide-out cart drawer for adding, updating, and removing items.
  • Stripe Integration: Securely handles credit card payments in a test environment.
  • Dynamic Homepage: Includes an autoplaying product carousel and category showcase.
  • Django Admin: A full back-end to manage store data and orders.

Technology Stack

  • Python & Django: For back-end logic, models, and rendering.
  • HTML5 & CSS3: For structure, layout, and responsive design.
  • JavaScript (ES6): For interactivity, cart drawer, and carousel.
  • Swiper.js: Used for touch-friendly product sliders.
  • Stripe API: Handles secure test payments.
  • SQLite3: Development database for data persistence.
