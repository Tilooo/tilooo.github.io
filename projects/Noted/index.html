About This Project

"Noted" was a comprehensive journey into full-stack web development. The goal was to build a complete, production-ready e-commerce application from scratch, focusing on a beautiful user interface and a robust back-end. The project covers the entire customer journey, from browsing products in a dynamic carousel, to a seamless "slide-out" cart experience, and finally to a secure payment process powered by Stripe. It was built with patience, creativity, and a lot of guidance from Google's AI, Gemini.

Key Features

  • Glassmorphism UI: A stunning "frosted glass" design with a custom yellow-and-green theme.
  • Full E-commerce Flow: Includes product listings, detail pages, search, categories, and a full checkout.
  • Interactive Cart: A modern slide-out cart drawer for adding, updating, and removing items.
  • Stripe Payment Integration: Securely handles credit card payments in a test environment.
  • Dynamic Homepage: Features an autoplaying product carousel and "Shop by Category" section.
  • Django Admin: A powerful back-end for managing all store data, including products, categories, and orders.

Technology Stack

  • Python & Django: For the entire back-end logic, database models, and server-side rendering.
  • HTML5 & CSS3: For structure and all the custom styling, animations, and responsive layouts.
  • JavaScript (ES6): For client-side interactivity, including the cart drawer and product carousel.
  • Swiper.js: A powerful library used to create the beautiful, touch-friendly product slider.
  • Stripe API: Integrated for secure and realistic payment processing.
  • SQLite3: The development database for storing all application data.
