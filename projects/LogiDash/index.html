Key Features

Everything you need to manage your supply chain efficiently.

Real-time Dashboard

Get an instant overview of your inventory, supplier stats, and category distribution with interactive charts and counters.

Demand Forecasting

Leverage Facebook's Prophet library to predict future product demand based on historical data trends.

Interactive Map

Visualize supplier locations and logistics routes on a dynamic map interface powered by Folium.

Kanban Board

Manage tasks and workflows visually with a drag-and-drop Kanban board for efficient project tracking.

Data Import/Export

Seamlessly upload and process data from CSV, Excel, and JSON files to update your inventory instantly.

Dark Mode

Fully supported dark mode with adaptive glassmorphism styles for comfortable viewing in any environment.
