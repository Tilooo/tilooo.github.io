Core Features

Discover the advanced functionality we built into Esme.

Live On-Map Routing

Calculates and displays the optimal driving route from your location to any point of interest, drawn directly on the map.

Interactive Point Discovery

Long-press anywhere on the map to instantly reveal the real-world address of that spot using reverse geocoding.

Dynamic Search & Filtering

Filter points by category and search within a visually represented radius that updates in real-time on the map.

Customizable Map Styles

Instantly switch between multiple detailed map styles, including standard, topographic, and cycling-focused views.

About The Project

Esme is an interactive mapping application that combines the power of Python for backend data processing and API services with Flutter for a beautiful, high-performance frontend interface.

This project was developed to explore full-stack application architecture and to make geographic data more accessible and engaging for users.
