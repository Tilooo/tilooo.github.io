Voice Assistant Project

A modern web-based voice assistant powered by Python

Overview

The Voice Assistant is a web-based application that combines Python's powerful backend capabilities with a modern web interface. It allows users to interact with their computer using voice commands, making everyday tasks more accessible and efficient.

Key Features

  • Voice command recognition and natural language processing
  • Web-based interface using Eel for Python-JavaScript integration
  • Real-time response to voice inputs
  • Custom command implementation capability
  • Integration with various APIs for enhanced functionality

Technology Stack

Python Eel HTML5 CSS3 JavaScript Speech Recognition API

Installation Guide

Prerequisites

  • Python 3.7 or higher
  • pip package manager

Setup Instructions

git clone https://github.com/Tilooo/personalAssistant.git
cd personalAssistant
pip install -r requirements.txt
python main.py

Usage Examples

Here are some example commands you can try with the Voice Assistant:

  • "What's the weather like today?"
  • "Open Google Chrome"
  • "Set a timer for 5 minutes"
  • "Play music on YouTube"
